
# Parallax

Welcome to the Parallax project!

## Overview
Parallax is a unified workspace for modern collaborative development, featuring:
- Modular backend (Java Spring Boot)
- Frontend (React/TypeScript)
- Python runner modules
- Design assets and prototypes

## Structure
- **Parallax-Backend**: Java Spring Boot backend for core business logic, REST APIs, authentication, and real-time collaboration.
- **Parallax-Frontend**: React/TypeScript frontend for user interface, dashboard, workspace, and interactive code blocks.
- **parallax-python-runner**: Python modules for code execution, collaboration, and integration with backend services.
- **Design**: Figma assets, UI prototypes, and additional module files.

## Features
- Real-time collaborative editing and code sharing
- Secure authentication and authorization
- Modular architecture for easy extension
- Interactive code blocks and workspace management
- Notification system and voice collaboration context
- Responsive UI with modern design

## Getting Started
1. Clone the repository:
   ```
   git clone https://github.com/Animesh-86/Parallax.git
   ```
2. See module-specific README files for setup and instructions.
3. Backend: Navigate to Parallax-Backend/backend and run with Maven.
4. Frontend: Navigate to Parallax-Frontend/frontend and run with npm/yarn.
